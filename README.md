# ✈️ TripMate

A serverless travel planner built using AWS services to help users securely manage and organize their trips.

---

## 🚀 Features

- 🔐 Secure User Authentication
- 👤 User Registration & Login
- ✈️ Trip Planning
- ☁️ Serverless Architecture
- ⚡ Fast API Responses
- 📱 Responsive Design

---

## 🛠️ Tech Stack

### Frontend

- HTML
- CSS
- JavaScript

### Backend

- AWS Lambda

### Cloud Services

- Amazon API Gateway
- Amazon Cognito
- Amazon DynamoDB

---

## ☁️ AWS Architecture

User
↓
Amazon Cognito
↓
API Gateway
↓
AWS Lambda
↓
DynamoDB
